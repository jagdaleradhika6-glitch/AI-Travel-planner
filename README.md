# 🌍✈️ AI Travel Planner

An intelligent AI-powered travel planning web application that generates personalized itineraries based on user preferences, budget, and interests. Designed with a modern UI and smooth user experience.

---

## 🔗 Live Demo

👉 **Live Link:** https://jagdaleradhika6-glitch.github.io/AI-Travel-planner/  
👉 **GitHub Repo:** https://github.com/jagdaleradhika6-glitch/AI-Travel-planner

---

## 🚀 Features

### 🏝️ Travel Planning System
- AI-based itinerary generation (simulated)
- Personalized trip planning
- Budget-based recommendations
- Multi-day travel schedule

### 📝 Smart Planner Form
- Destination input with suggestions  
- Date selection with validation  
- Travelers count slider  
- Budget range control  
- Travel preference selection  

### 📅 AI Itinerary Output
- Day-wise travel plan  
- Activity scheduling (morning, afternoon, evening)  
- Cost estimation  
- Travel tips & recommendations  

### 🌐 UI & Experience
- Fully responsive design  
- Smooth scrolling navigation  
- Animated sections & transitions  
- Beautiful gradients and layout  

### 🗺️ Explore Section
- Popular destinations (Paris, Tokyo, Bali, etc.)  
- Ratings, pricing, and descriptions  
- Interactive cards with hover effects  

### ⭐ Testimonials
- Customer reviews with ratings  
- Clean UI grid layout  

---

## 🛠️ Tech Stack

- **Frontend:** React.js (CDN)  
- **Styling:** Tailwind CSS  
- **Language:** JavaScript (ES6)  
- **Icons:** Lucide React  
- **Fonts:** Google Fonts
